Mes Projets

Développement d'un back-end pour un site de livres

J'ai développé le back-end d'une application en ligne permettant aux utilisateurs de gérer leur collection de livres, d'ajouter des critiques, et de consulter les recommandations basées sur leurs lectures. Le projet a été réalisé avec Node.js, Express, et MongoDB pour gérer les données des utilisateurs, des livres et des interactions.

Problématiques rencontrées : Assurer une gestion efficace des données utilisateur et des critiques, tout en maintenant la performance de l'application. J'ai surmonté cela en implémentant une architecture RESTful et en optimisant les index de la base de données pour des recherches rapides.

Compétences développées : Conception d'API REST, gestion de bases de données NoSQL avec MongoDB, authentification utilisateur avec JWT, et optimisation des requêtes.

Booki - Intégration d'un site d'agence de voyage

J'ai réalisé l'intégration d'une maquette pour un site d'agence de voyage nommé Booki en utilisant uniquement HTML et CSS. Ce projet permet aux utilisateurs de rechercher des hébergements et des activités touristiques avec une mise en page responsive adaptée aux différents appareils (ordinateurs, tablettes, mobiles).

Problématiques rencontrées : Adapter la mise en page pour différents formats d'écran et optimiser les performances de chargement des images. J'ai surmonté cela en utilisant des techniques de responsive design comme Flexbox et Grid, et en compressant les images pour améliorer les temps de chargement.

Compétences développées : Maîtrise du responsive design, utilisation de CSS Flexbox et Grid, optimisation des performances web, intégration de maquettes.

Kasa - Site de location d'appartement

J'ai développé une application web pour la location d'appartements nommée Kasa, utilisant React pour la création d'une interface utilisateur dynamique. Le site permet aux utilisateurs de consulter les annonces d'appartements, de filtrer les résultats selon leurs besoins, et de consulter les détails des propriétés.

Problématiques rencontrées : Gestion de l'état des composants dans une application à plusieurs pages et optimisation du rendu pour assurer une expérience utilisateur fluide. J'ai utilisé React Router pour la navigation et mis en place une gestion centralisée de l'état avec des hooks pour améliorer les performances.

Compétences développées : Développement d'applications React, gestion de l'état avec des hooks, navigation avec React Router, optimisation des performances dans une application Single Page (SPA).
